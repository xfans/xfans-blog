Linux开启虚拟AP

By xfans

October 9, 2017

安装create_ap

pacman -S create_ap

增加一个虚拟网络接口并设置一个mac地址

sudo iw dev wlp3s0 interface add wlp3s0_ap type __ap

sudo ip link set dev wlp3s0_ap address 04:e2:b9:17:18:72

创建AP

sudo create_ap -c 11 wlp3s0_ap wlp3s0 xfans_ap 321321321

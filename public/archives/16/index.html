Linux Manjaro安装kernel及Kernel Headers

2017-09-29 (Last Modified: 2017-09-29)
Linux

查找可用内核

pacman -Ss linux-headers

安装

pacman -S linuxXY linuxXY-headers

注意:XY为你要安装的版本号,如:

pacman -S linux49 linux49-headers
